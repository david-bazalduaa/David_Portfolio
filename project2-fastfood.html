Healthy Fast Food Optimizer

This project tackles a common challenge in modern lifestyles: finding healthy food options in fast food environments. Through extensive research and data analysis, we developed a recommendation system that helps users build nutritious meal combinations from fast food menus, while respecting their dietary goals and restrictions. We began by exploring and collecting nutritional datasets from Kaggle, which included detailed information about calories, protein, carbohydrates, fat, and sodium content for various menu items across major fast food chains. To complement this, we also conducted manual data curation from platforms like Uber Eats, Rappi, and official restaurant websites, ensuring up-to-date and realistic food selections for the optimizer. The core of the project relies on linear programming. We formulated an optimization model using scipy.optimize.linprog to identify combinations of food items that meet specific nutritional requirements while minimizing total calories or cost. Users can filter based on preferences (e.g., vegetarian), set calorie limits, or avoid certain macronutrients. The project combines applied mathematics, real-world nutrition, and user experience design, offering a data-driven alternative to navigating fast food in a smarter way.

Project Highlights

  • Goal: Suggest healthy meal options using optimization techniques.
  • Approach: Filter food items by user preferences, then solve for nutritional constraints using scipy.optimize.linprog.
  • Interface: Built a Streamlit web app for an interactive user experience.


Technologies Used

  • Python, Pandas, NumPy
  • Linear Programming (SciPy)
  • Excel (Fast Food Nutrition Dataset)
