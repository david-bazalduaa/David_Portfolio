Banknote Authentication Using PCA

In this project, we explored a classic machine learning problem: classifying banknotes as authentic or forged based on their statistical features. The dataset contained values extracted from images of banknotes such as variance, skewness, kurtosis, and entropy. We applied Principal Component Analysis (PCA) to reduce dimensionality and enhance model interpretability. After transforming the original data into principal components, we trained a classifier to distinguish between genuine and counterfeit banknotes. The project highlights how unsupervised techniques like PCA can aid in preprocessing and visualizing high-dimensional data, especially for binary classification problems.

Key Features

  • Dimensionality reduction using PCA (from 4D to 2D for visualization).
  • Classification of banknotes using transformed features.
  • Visualization of decision boundaries in principal component space.
  • Evaluation of model performance after transformation.


Technologies Used

  • Python (NumPy, Pandas, Scikit-learn, Matplotlib)
  • Jupyter Notebooks
  • Unsupervised learning (PCA)

Principal Component Analysis (PCA)

The scatter plots above represent combinations of the first six principal components (Y1 through Y6) derived from the original banknote dataset. Each dot corresponds to a sample: red for forged banknotes and green for genuine ones. The strong visual separation in Y1 vs Y2 indicates the power of PCA in dimensionality reduction and classification.

Variance Contributions by Component

  • Y1: Explains 66.75% of the variance – the most influential component.
  • Y2: Adds 20.82% to the explanation. Together with Y1, they account for 87.57%.
  • Y3: Adds another 5.42%, bringing the cumulative to 92.98%.
  • Y4 to Y6: Each explains less than 5% and can be excluded with minimal information loss.

Based on this analysis, using only the first two components (Y1 & Y2) is highly effective for visualizing and modeling the dataset, as they capture most of the variability with minimal complexity. This reduction simplifies downstream classification without sacrificing accuracy or interpretability.
