IMSS Predictive Cases

This project focuses on predicting morbidity cases in the Mexican Social Security Institute (IMSS) using historical data and neural networks. It integrates machine learning with data visualization to support public health strategy and decision-making.

Overview

  • Goal: Forecast morbidity trends for various medical conditions using structured historical data.
  • Approach: Data cleaning, time-series transformation, and modeling with a neural network using Keras.
  • Output: Visual forecasts integrated in an interactive Power BI dashboard.

Technologies Used

  • Python, Pandas, Keras
  • Power BI
  • Jupyter Notebooks
