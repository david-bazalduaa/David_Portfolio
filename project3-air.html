Air Pollution Contingency Prediction

This project focuses on analyzing air pollution data in Mexico City (CDMX) to detect and predict environmental contingencies based on historical trends. Using official data from SEDEMA, we investigated the behavior of ozone (O3) and NOx pollutants across different monitoring stations and times. The analysis included temporal and spatial correlations, hourly peak detection, and the use of classification models such as Support Vector Machines (SVM) to determine patterns that indicate the likelihood of a contingency occurring. The methodology included feature engineering, correlation heatmaps, scatter plots with lags, and SVM classification on interpolated and preprocessed data. All insights were visualized using Python, and summarized in an interactive Power BI dashboard.

Key Features

  • Exploratory Data Analysis on pollutant concentrations (O3, NOx).
  • Hourly ozone maximum detection across zones in CDMX.
  • Correlation matrix and lagged scatter plots for deeper relationships.
  • Custom contingency labeling based on SEDEMA criteria.
  • SVM model trained per zone to predict contingency likelihood.


Technologies Used

  • Python (Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib)
  • Jupyter Notebook
  • Support Vector Machines (SVM)

SVM Classification per Monitoring Station

The image above shows the results of applying a Support Vector Machine (SVM) with an RBF kernel across different pollution monitoring stations in Mexico City. Each subplot represents a station, where we trained a model to classify days with potential pollution contingencies based on NOx and Ozone (O3) levels.

Data was first standardized and interpolated to fill gaps, then used to create binary labels based on threshold values inspired by SEDEMA regulations. The brown contour zones represent regions where the SVM predicts the likelihood of a contingency.

This spatial and pollutant-specific approach allows us to understand how local pollution dynamics differ by region, providing a valuable tool for targeted environmental policy and alert generation.
